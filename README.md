🛒 MERN E-Commerce Store

A full-stack E-Commerce Web Application built using the MERN Stack (MongoDB, Express, React, Node.js). This project provides a complete online shopping experience with user authentication, product management, and order functionality.

---

🚀 Features

- 🔐 User Authentication (Login/Register)
- 🛍️ Product Listing & Filtering
- 🛒 Add to Cart Functionality
- 💳 Checkout Process
- 📦 Order Management
- 🧑‍💼 Admin Panel (Manage Users, Products, Orders)
- 📱 Fully Responsive UI

---

🛠️ Tech Stack

Frontend:

- React.js
- Redux Toolkit
- Tailwind CSS / CSS

Backend:

- Node.js
- Express.js

Database:

- MongoDB (Mongoose)

---

📁 Project Structure

root/
│── backend/       # Express server & APIs
│── frontend/      # React application
│── .env           # Environment variables
│── package.json   # Root dependencies

---

⚙️ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

2️⃣ Install dependencies

Backend

cd backend
npm install

Frontend

cd frontend
npm install

---

🔑 Environment Variables

Create a ".env" file in the backend folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

---

▶️ Run the Project

npm install

cd frontend
npm i

npm run dev
